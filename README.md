# Plants: REST API для обработки и предоставления данных о растениях

## Описание

Plants - это Java приложение, предоставляющее REST API для парсинга файлов с информацией о растениях и последующей передачи этих данных на фронтенд. Приложение разработано для удобного управления и отображения данных о растениях в клиентском приложении.

## Функциональность

•   **Парсинг файлов:**  Приложение принимает файлы (например, CSV, JSON, TXT) с данными о растениях и извлекает необходимую информацию.
•   **Обработка данных:**  Полученные данные обрабатываются и приводятся к единому формату, удобному для передачи на фронтенд.
•   **REST API:**  Приложение предоставляет REST API для доступа к данным о растениях.  Основные эндпоинты позволяют получать списки растений, информацию о конкретном растении и выполнять другие операции.
•   **Обработка ошибок:**  Предусмотрена обработка ошибок при парсинге файлов и работе API, обеспечивающая стабильность работы приложения.

## Технологии

•   **Java:**  Основной язык программирования.
•   **Spring Boot :** Для упрощения разработки REST API и управления зависимостями.
•   **Jackson:**  Для сериализации и десериализации данных в формат JSON.
•   **Maven:**  Система сборки и управления зависимостями.

## Структура проекта
•   **config/** – Конфигурационные файлы для Cors
•   **controllers/** – контроллеры
•   **Parcing/** - классы необходимые для обработки данных
•   **test/** - классы для теста отправки и получения запросов

Для запуска приложения необходимо в аргументах запуска указать путь к данным. Пример: java -jar -Ddata.folder="Путь до папки" "название.jar"
